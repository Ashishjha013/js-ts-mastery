## A calm learning path

This repo is arranged as a sequence of small, readable milestones. Each folder is meant to be explored slowly, edited lightly, and understood completely before moving on.

## Curriculum

| Stage                                    | Focus                                           |
| ---------------------------------------- | ----------------------------------------------- |
| [01-js-fundamentals](01-js-fundamentals) | Core language patterns and everyday JavaScript  |
| [02-node-basics](02-node-basics)         | Server-side foundations and runtime thinking    |
| [03-express-basics](03-express-basics)   | Routing, requests, responses, and app structure |
| [04-authflow-v0](04-authflow-v0)         | Authentication flow practice and integration    |

## How to use this repo

1. Start from the first stage and work forward.
2. Read the code before changing the code.
3. Make one small improvement at a time.
4. Run it, observe it, then refine it.

## Guiding principles

- Keep examples small.
- Favor clarity over cleverness.
- Let each folder stand on its own.
- Aim for understanding, not just completion.
