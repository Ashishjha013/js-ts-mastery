# JS & TS Mastery

Minimal notes, practical exercises, and focused builds for learning JavaScript, Node.js, Express, and auth flows.

Updated for July 2026.

## What this repo is

This workspace is organized as a progressive learning path. Each folder is a small milestone, and each day is meant to be read, edited, and rebuilt until the idea feels natural.

## How to use it

1. Start with the earliest day in the path.
2. Read the code before changing it.
3. Make one small improvement at a time.
4. Run the file, inspect the result, and repeat.

## Style

The goal here is not volume. It is clarity, repetition, and enough practice to make the fundamentals stick.

## Notes

- Keep examples small.
- Prefer understanding over memorizing.
- Let each folder stay self-contained.
